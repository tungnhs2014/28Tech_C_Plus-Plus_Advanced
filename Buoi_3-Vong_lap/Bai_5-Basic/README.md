# Bài 5: Chữ số đầu tiên và cuối cùng

## Mô tả bài toán  
Cho số nguyên không âm N, bạn hãy in ra chữ số đầu tiên và cuối cùng của N bằng cách sử dụng vòng lặp while

### Input Format
- Dòng duy nhất chứa số nguyên `N` 

### Constraints
- `0 <= N <= 10^18`

### Output Format
In ra đáp án của bài toán.

### Sample Input 0
```
28282828
```
### Sample Output 0
```
2 8
```
