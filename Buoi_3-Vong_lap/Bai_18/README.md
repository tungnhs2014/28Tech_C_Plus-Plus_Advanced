# Bài 18: Đếm chữ số nguyên tố của số nguyên

## Mô tả bài toán
Nhập vào n nguyên. Đếm số lượng chữ số của n là số nguyên tố.

### Input Format
Số nguyên không âm n

### Constraints
- `0 <= N <= 10^18`

### Output Format
Kết quả của bài toán

### Sample Input 0
```
1222333999888
```
### Sample Output 0
```
6
```
