# Bài 16: Đếm số lượng chữ số của N

## Mô tả bài toán
Nhập vào n, đếm số lượng chữ số n và in ra kết quả. Lưu ý số 0 là số có 1 chữ số.

### Input Format
Số nguyên không âm n

### Constraints
- `0 <= N <= 10^18`

### Output Format
Số lượng chữ số của n

### Sample Input 0
```
123456789
```
### Sample Output 0
```
9
```
