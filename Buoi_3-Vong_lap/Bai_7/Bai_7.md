# Bài 7: Liệt kê ước

## Mô tả bài toán
Đếm số lượng ước và liệt kê các ước theo thứ tự tăng dần của số nguyên dương N.

### Input Format
- Số nguyên dương `N` 

### Constraints
- `1 <= N <= 10^6`

### Output Format
Kết quả của bài toán

### Sample Input 0
```
28
```
### Sample Output 0
```
6
1 2 4 7 14 28
```
