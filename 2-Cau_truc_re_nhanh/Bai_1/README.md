# Bài 1: Tính giá trị của biểu thức 1

## Mô tả bài toán  
Cho biểu thức `A(x) = x^3 + 3x^2 + x + 1` với giá trị của x được nhập từ bàn phím, tính và in ra giá trị của biểu thức trên

---

## Input Formant 
- Số nguyên `x`.  

## Constraints  
- `-10^5 <= x <= 10^5`  

---

## Output Format  
- In ra kết quả của biểu thức

---

## Sample Input 0
    2

## Sample Output 0
    23
